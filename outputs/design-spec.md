# 어린이용 웹 계산기 UI/UX 설계서

> 타겟 사용자: 초등학교 저학년 (7-9세)
> 작성일: 2026-05-10

---

## 1. 디자인 원칙

### 어린이 친화적 UI 원칙
- **큰 터치 영역**: 버튼 최소 크기 60x60px
- **밝고 친근한 색상**: 파스텔톤 + 주요 색상 강조
- **명확한 피드백**: 버튼 클릭 시 시각적 반응
- **격려하는 톤**: 오류 시에도 부드럽고 긍정적인 메시지
- **단순한 레이아웃**: 한 화면에 하나의 주요 기능

## 3. 폰트 가이드

- **주요 폰트**: 'Jua', sans-serif (한글 친화, 둥근 느낌)
- **숫자 표시**: 36px (계산 결과)
- **버튼 텍스트**: 24px
- **안내 메시지**: 18px

## 5. UX 플로우

### 5.1 계산기 플로우
```
시작 -> 숫자 입력 -> 연산자 선택 -> 숫자 입력 -> = 클릭 -> 결과 표시
                                                          |
                                            C 클릭 -> 초기화
```

### 5.2 퀴즈 플로우
```
퀴즈 탭 클릭 -> 랜덤 문제 표시 -> 답 입력 -> 정답 확인 클릭
                                              |
                               +----+---------+----------+
                               |                         |
                           정답일 때              오답일 때
                               |                         |
                    "잘했어요!" + 점수 증가    "다시 도전해봐!" 표시
                               |                         |
                               +----+-----+--------------+
                                    |
                            [ 새 문제 ] 클릭 -> 다음 문제
```

## 6. 컴포넌트 상세

### 6.1 숫자 버튼
- 크기: 70x70px
- 모서리: border-radius 15px
- 배경: #4ECDC4
- 호버: 밝기 10% 증가
- 클릭: scale(0.95) 애니메이션

### 6.2 연산자 버튼
- 크기: 70x70px
- 모서리: border-radius 15px
- 배경: #FF6B6B
- 폰트: 28px, bold

### 6.3 디스플레이
- 높이: 80px
- 배경: 흰색
- 테두리: 2px solid #E0E0E0
- 모서리: border-radius 10px
- 텍스트: 오른쪽 정렬, 36px

### 6.4 탭 버튼
- 활성: 배경 #4ECDC4, 텍스트 흰색
- 비활성: 배경 #E8E8E8, 텍스트 #666

## 7. 피드백 메시지

### 정답 시
- 텍스트: "잘했어요!", "대단해요!", "정답이에요!"
- 색상: #2ECC71
- 애니메이션: 텍스트 살짝 커졌다 작아지는 효과

### 오답 시
- 텍스트: "다시 도전해봐요!", "조금만 더 생각해봐요!"
- 색상: #F39C12
- 톤: 절대 부정적이지 않게

---

## 8. Dev 전달 사항

1. **단일 HTML 파일**로 구현 (CSS, JS 인라인)
2. **반응형 불필요** - 데스크톱 기준 고정 너비 (400px)
3. **외부 의존성 없음** - 순수 HTML/CSS/JS
4. **퀴즈 문제**: 1~20 범위의 덧셈/뺄셈 (저학년 수준)
5. **점수 저장**: 페이지 새로고침 시 초기화 가능 (localStorage 불필요)
